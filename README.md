# Browser extension sample

A simple browser extension skeleton for Chrome-based browsers and Firefox.

Features:
 * Popup sample
 * Content script sample
 * Background event handler sample
 * Logger module
 * ESLint integration
 * Webpack config
 * GitHub Actions workflows

## Usage

Development:
```
$ npm install
$ npm run start
# or for Firefox:
$ npm run start-ff
```
